# 📒 Contact Management System (C++)

A simple console-based Contact Management System built in C++.
It allows users to manage contacts with features like add, search, edit, and delete.

---

## 🚀 Features

* ➕ Add Contact
* 📋 Show All Contacts
* 🔍 Search Contact
* ✏️ Edit Contact
* ❌ Delete Contact
* 💾 Save/Load data from file (contacts.txt)

---

## 🛠️ Technologies

* C++
* File Handling
* OOP (Object-Oriented Programming)

---

## 📂 Project Files

* MAIN.CPP
* CONTACT.CPP
* CONTACT.H
* CONTACTBOOK.CPP
* CONTACTBOOK.H
* contacts.txt

---

## ▶️ How to Run

1. Compile:

   ```
   g++ MAIN.CPP CONTACT.CPP CONTACTBOOK.CPP -o app
   ```
2. Run:

   ```
   ./app
   ```
